☁️ 自動クラウド同期 (Firebase)
保存するたびに自動でクラウドへ送信。
他のデバイスで開くと30秒以内に自動反映。
データはタイムスタンプで管理し古いデータで上書きされません。
Firebase無料プラン: 1GB保存 / 10GB転送/月
Firebase Realtime Database URL
⚠️ ファイルを直接開いています（file://）。Firebase同期はGitHub PagesのURLで開いた場合のみ動作します。
⚠️ HTTP 423 = Firebaseのルールがロックされています
以下の手順で1分で解決します
① console.firebase.google.com を開く
② 該当プロジェクト → 左メニュー「Realtime Database」
③ 上部タブ「ルール」をクリック
④ 内容を以下に書き換えて「公開」を押す
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
※90日で期限切れになりますが、同じ操作で延長できます
✓ Firebase接続済み。自動同期が有効です。
Firebase 無料設定手順（5分）
① Googleアカウントで Firebase にアクセス
　console.firebase.google.com → 「プロジェクトを作成」
　→ 名前「nyano」→ Googleアナリティクス: OFF → 作成
② Realtime Database を作成（重要）
　左メニュー「構築」→「Realtime Database」
　→「データベースを作成」ボタンを押す
　→ ロケーション: asia-southeast1（シンガポール）を選択
　→「テストモードで開始」→「有効にする」
③ URLをコピーして貼り付け
　データベース画面上部のURLをコピー
　→ シンガポール選択時のURL形式 (`.app`で終わる):
　https://プロジェクトID-default-rtdb.asia-southeast1.firebasedatabase.app
　→ アメリカ選択時のURL形式 (`.com`で終わる):
　https://プロジェクトID-default-rtdb.firebaseio.com
　→ 上の入力欄に貼り付けて「保存」→「接続テスト」で確認
完成！PCとスマホ両方で同じURLを設定すれば自動同期
※テストモードは90日で期限切れ。「接続テスト」で423エラーが出たら画面の手順通りにRulesを変更してください
